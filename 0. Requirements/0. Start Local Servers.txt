1. Avvio del Backend (Docker + Supabase)
Basta far partire Docekr Desktop (così fa partire supabase start)

2. Avvio del Backend (Python/Edge Functions)
supabase functions serve
e
python api/index.py

3. Avvio del Frontend (React + Vite)
npm run dev
(se serve forzare la cancellazione della cache: Remove-Item -Recurse -Force .next)


4. Esecuzione dei Test (Vitest)
- Modalità Interattiva (Watch): Ricarica i test automaticamente ad ogni modifica.
npx vitest

- Esecuzione Singola: Ottimo per CI/CD o verifiche rapide.
npx vitest run
